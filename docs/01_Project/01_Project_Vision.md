# Overview

NexHome is a Smart Home IoT platform built using a Microservices architecture to simulate a production-inspired connected home ecosystem.

The project focuses on demonstrating modern Software Development Engineer in Test (SDET) practices including API testing, UI automation, SQL validation, integration testing, contract testing, performance testing, and resilience testing.

Rather than being only an IoT application, NexHome serves as a complete software engineering portfolio project showcasing the entire software development lifecycle.

---

# Vision Statement

To build a production-inspired Smart Home IoT platform that demonstrates modern microservices architecture together with professional software testing methodologies and Agile software development practices.

---

# Business Goal

Develop a realistic software system that demonstrates the design, development, testing, and maintenance of distributed applications while following industry-standard engineering workflows.

---

# Success Criteria

The project will be considered successful when:

- All planned sprint objectives are completed.
- All microservices communicate successfully.
- Every feature is fully tested.
- Sprint documentation is completed.
- Git history reflects incremental development.
- Jira accurately tracks project progress.
- End-to-end workflows execute successfully.

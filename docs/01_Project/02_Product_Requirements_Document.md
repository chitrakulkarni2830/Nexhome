# Product Summary

NexHome is a Smart Home IoT platform consisting of multiple services that work together to manage connected smart devices.

The system allows users to:

- Register devices
- View device status
- Receive live telemetry
- Control smart devices
- Create automation routines
- View analytics
- Monitor device health

---

# Project Objectives

- Demonstrate Microservices Architecture
- Implement REST APIs
- Build a responsive frontend
- Practice API communication
- Showcase database validation
- Demonstrate software testing practices
- Follow Agile Scrum methodology

- FastAPI backend
